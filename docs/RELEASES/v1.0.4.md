# Release Notes v1.0.4 - Group Qualification & GCP Official Deploy

This release marks the officialization of the local development version to the Google Cloud Console (Cloud Run) and the activation of the new "Group Qualification" logic.

## Key Features

### 1. Group Qualification System
- **Clients**: Added a new mandatory section during the quote request process (Families/Friends, number of children, friends composition, pets).
- **Owners**: Implemented a "Group Analysis" dashboard in the `OwnerConfirmationView.jsx` that calculates a **Reliability Score** (Reliability Scoring) based on group characteristics.

### 2. Database & API
- **Supabase**: Added the `group_details` JSONB column to the `quotes` table to persist structural group analysis data.
- **RPC/SQL**: Successfully migrated remote schema to support version 1.0.x features.

### 3. Infrastructure & Deployment
- **Google Cloud Platform**: Version 1.0.4 is now "Officialized" via GitHub -> Cloud Build trigger.
- **Vite/React**: Aligned local and remote versions to version 1.0.4.

## Technical Details
- **Build ID**: Triggered via `ace1537` commit.
- **DB Migration**: Applied via `ALTER TABLE quotes ADD COLUMN IF NOT EXISTS group_details JSONB;`.
