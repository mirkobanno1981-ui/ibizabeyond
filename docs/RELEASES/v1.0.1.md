# Ibiza Beyond - Release Notes v1.0.1 (04/04/2026)

## Sommario delle Modifiche
Questa versione si focalizza sul miglioramento dell'esperienza utente (UX) durante la fase di prenotazione delle ville, fornendo feedback immediati e chiarimenti sulle regole di soggiorno direttamente nel calendario.

## Nuove Funzionalità e Miglioramenti
### 1. Sistema di Validazione Calendario Dinamico
- **Validazione in Tempo Reale**: Aggiunta una logica avanzata nella selezione delle date che rileva immediatamente se le date scelte violano le regole della villa (ad es. soggiorno minimo non rispettato o giorno di check-in non valido).
- **Messaggi di Feedback**: Invece di mostrare solo "Data non valida", il sistema ora fornisce suggerimenti specifici come "Il soggiorno minimo per questo periodo è di 7 notti" o "Questa villa accetta solo check-in di sabato".

### 2. Banner Guida alla Prenotazione
- **Visualizzazione Regole**: Inserito un piccolo banner sopra il calendario delle ville che mostra dinamicamente i requisiti minimi di quella specifica villa.
- **Informazioni incluse**: Numero minimo di notti e giorni consentiti per il check-in, aggiornati in base alla stagione.

### 3. Sidebar di Prenotazione Ottimizzato
- **Feedback Continuo**: Il box di riepilogo nella sidebar ora mostra lo stato della selezione ("Seleziona una data di arrivo", "Seleziona la data di partenza") e i codici di errore descrittivi per guidare l'utente alla corretta prenotazione.

## File Modificati
- `src/components/VillaView.jsx`: Principali modifiche alla logica del calendario e dell'interfaccia utente.
- `package.json`: Incremento della versione a 1.0.1.
- `package-lock.json`: Aggiornamento metadati dipendenze locali.
