# Release notes - v1.0.2

## 🚀 Nuove Funzionalità: Qualificazione Gruppo & Scoring Affidabilità

Questa versione introduce un passo fondamentale nel processo di preventivazione: la **Qualifica dei Gruppi di Ospiti**. Questa funzionalità permette agli agenti di fornire dettagli cruciali ai proprietari delle ville, agevolando la loro decisione di accettazione tramite un calcolo automatico della "Reliability" (Affidabilità).

### 🛠 Modifiche Tecniche

#### 1. Frontend (VillaView.jsx)
- Aggiunto **nuovo modulo "Group Qualification"** nel Quote Modal.
- Logica condizionale per catturare dati specifici:
  - **Famiglie**: Numero di bambini.
  - **Amici**: Composizione (Soli uomini, Sole donne, Misto) e flag "Solo Coppie".
  - **Animali**: Toggle per la presenza di animali domestici.
- Reset automatico di tutti i campi al termine dell'operazione.

#### 2. Vista Proprietario (OwnerConfirmationView.jsx)
- Implementata la sezione **"Group Analysis"** visibile solo al proprietario.
- **Reliability Score**: Calcolo visivo dell'affidabilità basato sul tipo di gruppo:
  - **Famiglie**: Profilo "HIGH (85%)" - Segnale Verde.
  - **Coppie**: Profilo "RELIABLE (75%)" - Segnale Blu.
  - **Gruppi Misti/Amici**: Profilo "MODERATE (60%)" - Segnale Arancio.
- Visualizzazione immediata di icone descrittive per una rapida consultazione.

#### 3. Database (Supabase Migration)
- Creata migrazione SQL: `supabase/migrations/20260404_add_group_qualification.sql`.
- Aggiunta colonna `group_details` di tipo **JSONB** alla tabella `quotes` per una gestione flessibile dei dati del gruppo.
